== Description ==

Easy to set-up the some featured functions.

= Function List: =
* Term
	* Allow html tags in taxonomy description (pre_term_description)

* WooCommerce
	* Allow full width featured image in Shop page
	* Allow only one product in Cart
	* Change the add to cart button's text
	* Change 'Proceed to PayPal' Text on Checkout button
	* Change 'has been added to your cart' Message when added to cart successfully
	* Change 'View Cart' Button Text
	* Change 'Proceed to Checkout' Button Text
	* Change 'Update Cart' Button Text
	* Shipping Discount

* WordPress
	* Disable Email Notification when there is new version of the WordPress
	* Disable Emoji
